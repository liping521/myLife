# 考研分享

### 第一篇
```
整理信箱，找到两封旧信，是社科院宪行室的一位老师回的，贴过来，希望能对某些人有用。  
  
一  
你好：  
    很高兴能接到你的来信，欢迎你报考我的研究生。根据以往的情况，一般是10：1的比例，不过，
    每年能够5门课程都靠及格并且总分上起分线的很少，一般也只有1到2人。一般来说，只要单科成
    绩在60分以上，总分达到起分线，即便成绩不在第1名，破格录取的可能性也很大。至于能否向其
    他学校推荐，我所以前没有出现这种情况，但一旦出现这种情况，我想也是有可能的。不过前提是要
    符合教育部规定的录取标准。
    从往年的经验来看，绝大部分考生的外语成绩都不过关。所以，靠考研究生往往是比外语。今年就有
    一个专业成绩第一但外语不及格的考生无法被录取的例子。所以，如果你要报考，一定要考虑外语实
    力，至少应当及格。欢迎你继续来信。

二  
你好：  
    我看了你的成绩，你的外语和政治、宪法成绩都比较稳定，如果考我所的研究生，基本上没有什么问题。
    通常考我所的研究生，虽然人数达到10：1，但外语和政治及格的也就是10：1，所以，如果你愿意的话，
    欢迎你考我的研究生。
```
