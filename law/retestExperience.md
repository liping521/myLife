# 复试经验分享

```
1. 中国社会科学院研究生院法硕考研复试：首先，社科院法硕复试一般不刷人。
这个基本是真的。今年的考题也略微有些改动。往年是抽题，今年直接取消了。
大家的题目都一样。所以，分数相对也公平。面试也不难，但是不能表现的太
糟糕。所以，想在北京求学的，又想上一个不错的，又风险不大的，我觉得社
科院是个不错的选择。  
2. 社科院法硕复试流程：首先，具体的复试安排可以在中国法学网上找到，每一
年的复试时间都不太一样，但是不会相差太大，所以临近3月下旬的时候多关注
中国法学网和社科院研究生院的网站。在这里说一下面试的问题吧。面试分为英
语面试和专业课面试两部分。在英语面试部分，原则上要求全部用英文回答，所
以必要的准备还是要的。关于英文自我介绍的问题，在2016年的复试中，基本
没有哪个同学被问自我介绍的问题，基本都是按照自己手中的序号进去面试，直
接省去了自我介绍的环节。英语面试的内容基本同当年的最新的时事热点相关，
2016年就主要围绕着慈善法展开面试，几乎每个同学的英语面试都和慈善有关，
据此推测（大部分人都这么说），2016年的英语面试选取的是一篇较长的时事文
章，然后根据字数分为若干小段，每个同学选择其中的一段来考试。所以，2017
年的面试基本会沿用这样的模式，所以，临近复试前，多看一些时事方面的报道，
查阅一下相关词汇。  
3. 接下来是专业课面试，法硕专业课的面试和上午的笔试基本都是考察大家最基本的
法硕知识，不会出生僻的知识点让大家作答。例如，我今年面试的题目就是法理学
里面的法的本质和法的特征。这是很基础的题。所以专业课方面不要太紧张，正常
发挥就没问题。复习的时候抓住大的要点，细小的点可以不看，或者不要太关注，
抓主要的知识点记忆。除了专业课的知识，一般情况下，还会有一个小的案例题，
一般没有固定答案，按照自己所学的知识阐述就好，如果老师问了，或者要求用某
一学科的某个知识点作答，那么就按老师说的做就好了。  
4. 其它的问题会涉及方方面面，爱好啊，分数啊，本科专业什么的。这个大概准备下
就行，切忌死记硬背。老师们需要的是你的问题思路，而不是背诵！  
5. 着装方面，这个不要太随意就好，让老师觉得这是一个学生该有的样子就行，一般
情况下，不用穿正装。注意礼貌，注意自己面试时候的言谈举止。
```
